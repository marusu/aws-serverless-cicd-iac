# aws-serverless-cicd-iac

This repository demonstrates a production-oriented serverless architecture on AWS built with Terraform and deployed via GitHub Actions using OIDC authentication.

It includes:

- Infrastructure as Code (Terraform)
- CI/CD pipeline with Plan/Apply separation
- API Gateway + Lambda + DynamoDB
- Deployment strategy implementation
- State management with S3 and DynamoDB (planned)

This project is designed to simulate a real-world DevOps workflow for serverless applications.
